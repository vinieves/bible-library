Pontos importantes do projeto
PHP 8.3 obrigatório (Laravel 13)
MySQL em produção (não use SQLite na VPS)
Queue worker com Supervisor — sem isso o WhatsApp não envia
Não rode php artisan db:seed completo — cria usuário demo com senha password
Use php artisan make:filament-user + SQL para marcar is_admin = 1
O projeto ainda não tem Git remoto — o guia inclui envio via SCP do Windows e opção GitHub
Arquitetura final
mediamrkt.online      → Nginx → Laravel (Bible Library) + Supervisor
wpp.mediamrkt.online  → Nginx → Docker (Evolution + Postgres + Redis)
Atualizar o Bible Library não derruba a Evolution. Atualizar a Evolution não derruba o site — a sessão do WhatsApp fica no volume Docker.

Abra os arquivos .md no Cursor e execute linha por linha na VPS via SSH. Se quiser, no próximo passo posso adaptar os guias com o IP real da sua VPS e seu e-mail já preenchidos.
